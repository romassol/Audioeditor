Описание
Данное приложение является аудиоредактором

Требования
Python версии не ниже 3.5

Состав
Консольная версия: main.py
Аудиоплеер: audioplayer.py
Тесты:  test.py

Консольная версия
Описание команд:
    --file/-f - имя wav файла, который будет обрабатываться (по дефолту - обычный.wav)
    пример: python main.py -f arabella.wav
    --speed/-s - изменение скорости в заданное количество раз (от 0 не включительно до 2) с изменением высоты звука
    пример: python main.py -s 0.5
    --temp/-t - изменение скорости в заданное количество раз (от 0 не включительно до 2) без изменение высоты звука
    пример: python main.py -t 1.5
    --pitch/-p - изменение высоты звука на заданное количество полутонов (от -12 до 12)
    пример: python main.py -p 6
    --volume/-v - на сколько (в дб) будет увеличена громкость
    пример: python main.py -v 10
    --split/-spl - позиция в миллисекундах по которой обрабатываемый файл будет обрезан и запись в 2 новый файла
    пример: python main.py -spl 6000
    --join/-j - имя wav файла, с которым будет объединен обрабатываем файл, т.е. обрабатываемый файл + новый файл
    пример: python main.py -j arabella.wav
Справка по запуску: python main.py --help
Пример запуска: python main.py -s 2 -v 10 -p -10

Аудиоплеер
Справка по запуску: python audioplayer.py --help
Пример запуска: python audioplayer.py -f обычный.wav

Подробности реализации
В основе всего лежит класс AudioEditor, реализующий обработку звука аудиофайла с расширением wav с возможностью ускорения, замедления, разреза, склейки, изменения громкости, изменения высоты звука.
Класс AudioPlayer отвечает за проигрывание аудиофайла.
